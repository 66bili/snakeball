## 📜 About this project
SNAKEBALL是一个休闲网页小游戏，在游戏中，玩家可以操控一条由球体组成的小蛇、在屏幕范围内随意移动，<br>
小蛇的身体每隔随机的时间会进行重置：小蛇的移动速度、以及身体的延迟都会发生改变，<br>
在关卡时间里，屏幕内会出现各种各样的子弹球，玩家需要根据情况来“吃”或者“碰撞”它们，并获得一定的分数，以计入排行榜。<br>

## 🛠 Set up
1. 初始化node_modules，构建项目

   ```sh
   npm install
   ```

2. 启动Vue开发服务器

   ```sh
   npm run serve
   ```

3. 将项目打包成静态网页

   ```sh
   npm run build
   ```
